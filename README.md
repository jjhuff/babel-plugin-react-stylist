# babel-plugin-react-stylist

Merge style arrays and autoprefix

## Installation

```sh
$ npm install babel-plugin-react-stylist
```

## Usage

### Via `.babelrc` (Recommended)

**.babelrc**

```json
{
  "plugins": ["react-stylist"]
}
```

### Via CLI

```sh
$ babel --plugins react-stylist script.js
```
